## 매장 상세 정보 조회

| 메소드 | 경로               | 짧은 설명           |
| ------ | ------------------ | ------------------- |
| GET    | /stores/{storeIdx} | 매장 상세 정보 조회 |

#### 요청 헤더
```json
Content-Type: application/json
```
### 응답 바디
#### 매장 상세 정보 조회 성공
```json
{
    "status": 200,
    "message": "매장 상세 정보 조회 성공",
    "data": {
        "name": "커피세포",
        "address": "서울 동대문구 이문로 50",
        "photo": [
            {
                "photoUrl": "매장 사진 url1"
            },
            {
                "photoUrl": "매장 사진 url2"
            }
        ],
        "time": "운영시간, 휴무일",
        "number": "매장 전화 번호"
    }
}
```

#### 매장 상세 정보 조회 실패

```json
{
    "status": 404,
    "message": "매장 상세 정보를 찾을 수 없습니다.",
    "data": null
}
```

#### INTERNAL SERVER ERROR

```json
{
    "status": 500,
    "message": "서버 내부 에러",
    "data": null
}
```

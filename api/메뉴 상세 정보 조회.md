## 메뉴 상세 정보 조회

| 메소드 | 경로   | 짧은 설명 |
| ------ | ------ | --------- |
| GET   | /stores/{storeIdx}/menu/{menuIdx} | 메뉴 상세 정보 조회 |

#### 요청 헤더
```json
Content-Type: application/json
```
### 응답 바디
#### 메뉴 상세 정보 조회 성공
```json
{
	"status": 200,
    "message": "메뉴 상세 정보 조회 성공",
    "data": {
        "menu_idx": 1,
        "photo_url": "사진 url",
        "price": 3000
    }
}
```

#### 메뉴 상세 정보 조회 실패

```json
{
    "status": 404,
    "message": "메뉴 상세 정보를 찾을 수 없습니다.",
    "data": null
}
```

#### INTERNAL SERVER ERROR

```json
{
    "status": 500,
    "message": "서버 내부 에러",
    "data": null
}
```

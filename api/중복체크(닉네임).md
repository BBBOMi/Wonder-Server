## 중복체크(닉네임)

| 메소드 | 경로                     | 짧은 설명       |
| ------ | ------------------------ | --------------- |
| GET    | /users/check?nick={nick} | 닉네임 중복체크 |





### QueryString 설명

| Parameter | 설명               | 예시       |
| --------- | ------------------ | ---------- |
| nick      | 중복 체크할 닉네임 | nick=bring |





### 요청 헤더

```
Content-Type: application/json
```





### 응답 바디

#### 중복 체크 성공

```
{
    "status" : 204,
    "message" : "중복 체크 성공",
    "data" : null
}
```



#### 중복 체크 실패

```
{
    "status" : 400,
    "message" : "중복 체크 실패",
    "data" : null
}
```



#### INTERNAL SERVER ERROR

```
{
    "status": 500,
    "message": "서버 내부 에러",
    "data": null
}
```
